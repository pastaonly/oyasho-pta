# 非加入を回答された方への個別メール（再考のご案内）

**対象:** 入会承諾フォームで「入会しない」とご回答いただいたご家庭

**目的:** ご判断の理由を伺いつつ、PTAの新制度を改めてご説明し、再考の機会をお伝えする
**送付方法:** メール（個別送信）

---

## メール文案

```
件名：【尾山台小PTA】ご回答へのお礼と、改めてのご案内

◯◯◯◯ 様

このたびはPTA入会のご案内にご回答をいただき、誠にありがとうございました。

ご記入いただいた「入会しない」とのご回答、しかと承りました。
PTAは任意加入の団体であり、ご家庭のご判断を最大限尊重いたします。

そのうえで、もしよろしければ、ご判断の背景となった理由を
お聞かせいただけないでしょうか。
今年度よりPTAは大きく仕組みを変えており、もしも以前のPTAの印象や、
ご懸念の点が判断材料となっていらっしゃる場合は、
誤解を解いたり、ご懸念に直接お答えできる可能性があります。

お忙しいなかご無理にとは申しませんが、ご返信いただければ幸いです。


■ 改めてのご案内

新しい尾山台小学校PTAは、ご加入いただいても以下のように
**ご家庭にデメリットが生じない設計** となっております。

【加入しても、こうした「負担」は一切ありません】
　・委員・役員の強制的な選出
　・PTA会費の徴収
　・活動への参加義務

【ご家庭の年間負担】
　・学校運営必要経費 500円／年
　　※この500円は、PTAへのご加入の有無にかかわらず
　　　全家庭から徴収させていただきます
　　　（コピー代・保険料等の実費にあたるため）

つまり、加入しても・しなくても、ご家庭の負担額は変わりません。


■ 一方で、ご加入されない場合は以下を失います

新制度では、PTA会員のメリットを次の2点に集約しました。
ご加入されない場合、この2つが受けられなくなります。

　① PTA総合補償制度保険の対象外となります
　　お子様の活動中・登下校中等の事故への補償が受けられません。
　　（詳細は下記「保険のご説明」をご覧ください）

　② PTA総会での議決権がなくなります
　　PTAの予算・事業計画など、運営方針に対して
　　ご家庭の意思を反映できなくなります。

費用負担は変わらないなか、保険・議決権を「失う」かたちとなる点が、
ご加入されない場合の主なポイントです。


■ 補足：PTA総合補償制度保険について

PTA会員のお子様を対象とした、全国PTA連絡協議会が提供する団体保険です。

【補償の対象となる主な場面】
　・PTA活動中のお子様の事故・怪我
　・登下校中の事故
　・PTA関連行事中の事故

【主な補償内容（参考）】
　・死亡・後遺傷害見舞金：最大500万円
　・入院見舞金：5万円
　・通院見舞金：1万円　ほか

【保険料のご負担】
　保護者個別の追加負担はありません。
　学校運営必要経費 500円／年 のなかで運用しています。

詳細な補償範囲・限度額の資料は、ご入会後にお渡しいたします。


■ 再考とご返信のお願い

上記をご覧いただいたうえで、もし「それなら入会してみようか」と
お考えになる場合は、お気軽にご連絡ください。
再加入は随時可能です。

また、ご判断は変わらない場合でも、ご回答時の理由・ご懸念だけでも
お聞かせいただけますと、今後のPTA運営の参考とさせていただきます。

【ご返信先】
　メール：◯◯◯◯
　LINE：◯◯◯◯（PTA本部）

PTAについての詳細はこちら：
　https://pastaonly.github.io/oyasho-pta/

ご多忙のなか恐縮ですが、よろしくお願い申し上げます。
お子様にとって、よい1年となりますよう、PTA一同願っております。


尾山台小学校PTA 本部役員一同
```

## 送信時のチェック

- [ ] 各家庭の宛名・お子様名を差し込む
- [ ] PTA本部のメール／LINE等の連絡先を埋める
- [ ] 送信名義を役員ミーティングで決定（会長個人／PTA本部／連名）
- [ ] 件名の調整（必要に応じて「尾山台小学校PTA入会についてのご案内」等）
