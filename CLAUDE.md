### Notionへのアクセス方法

Notion MCP（`mcp__claude_ai_Notion__notion-*`）経由でページの閲覧・検索・作成・更新が可能です。Notion側で議論・検討中の内容を参照したい場合は、まず `notion-fetch` でルートページから辿ってください。

## リポジトリ構成

| パス | 内容 |
|---|---|
| `index.html` | 公開サイト（GitHub Pages: pastaonly/oyasho-pta） |
| `export/` | 令和8年5月改正の確定文書（規約・細則・契約書雛形等） |
| `reference/` | 参考資料・改革検討記録・旧規約 |
| `asobiba/` | 遊び場開放関連資料（区の規則・様式等） |
| `決議/` | PTA総会・臨時総会の決議結果 |
| `3年生執行委員-運営ルール/` | 業務マニュアル（行事受付・旗振り等） |
| `tmp/` | 作業用スクリプト・タスクボード等 |
| `issues.md` | 未解決の検討課題一覧 |

## 役割分担の目安

- **Notion**：流動的な議論・打ち合わせメモ・検討中のタスク・スケジュール
- **このリポジトリ**：確定文書・運営マニュアル・決議結果・公開サイト

矛盾が生じた場合は、ドキュメント先頭の「最終更新日」が新しいほうを優先。
