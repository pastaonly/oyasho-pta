## 2. 論点の整理

### 論点1：案1と案2のどちらがよいか

**案2（11名）を推奨。** 理由は以下の3点。

**（1）前提条件がカバーしている。** 全員が一つのLINEグループで動き、PTA役員・支援コーディネーターも入る体制であれば、各業務担当が1名でも孤立しない。困ったときにグループ内で助けを求められるので、「相談相手がいない」リスクが低い。

**（2）行事受付・旗振り・通学路点検は稼働が限定的。** 行事受付は年数回、旗振りは学期に1回、通学路点検は年1回。それぞれ専任で配置するほどの業務量ではない。旗振り管理と通学路点検は2名が兼務しあう体制とし、実際の作業は3年生保護者全体から都度募集する方が合理的。

**（3）選出の負担が減る。** 15名の確保は3年生保護者の規模によっては相当なハードル。11名なら選出がスムーズになり、「なり手がいない」問題のリスクを下げられる。

> ただし、案2でも遊び場委員6名の確保が最大の課題。遊び場は通期稼働なので、ここは人数を削れない。

### 論点2：副委員長の役割

副委員長を置き、以下の2つを明確な責務とする。

**① 委員長代理 ＋ 遊び場委員副委員長兼務**
執行委員長が不在のとき、PTA役員会との窓口を引き継ぐ。加えて、遊び場開放運営委員会の副委員長（委員長補佐・職務代理）を兼務する。

**② 引き継ぎ資料の作成**
年度末の引き継ぎ資料作成を主担当として担う。委員長が通期の統括に集中できるようにする。

| | 委員長 | 副委員長 |
|---|---|---|
| PTA役員会との窓口 | ◎ 主担当 | ○ 代理 |
| 各業務担当への相談対応 | ◎ | ◎（分担） |
| 遊び場開放運営委員会への出席 | ◎ | ○ 代理（副委員長として） |
| 遊び場委員長の代理 | — | ◎ 遊び場副委員長を兼務 |
| LINEグループの管理 | ◎ | ○ 補佐 |
| 引き継ぎ資料の作成 | ○ | ◎ 主担当 |

## 4. 情報共有の設計

### 4-1. LINEグループの構成

**2つのグループで運用する。** 遊び場開放は通期稼働でやり取りの量が多いため、専用グループを分離する。メイングループに遊び場の日常連絡が流れ続けると、行事受付・旗振り・通学路点検の連絡が埋もれてしまうため。

#### ① 執行委員会メイングループ

| 区分 | メンバー | 備考 |
|---|---|---|
| 執行委員 | 委員長・副委員長・行事受付担当2名・旗振り管理／通学路点検担当2名 | 6名 |
| 遊び場委員 | 遊び場委員長（※兼任の場合は重複なし） | 遊び場の代表として参加 |
| PTA役員 | 会長 or 副会長（少なくとも1名） | 予算・学校との調整サポート |
| 支援Co | 学校支援コーディネーター | 保護者・地域・学校の橋渡し |

**用途：** 全体連絡、行事受付・旗振り管理・通学路点検の業務連絡、PTA役員会への報告・相談、遊び場からの共有事項（月次報告レベル）

**運用ルール：**
- 業務連絡は「【行事受付】」「【旗振り管理】」「【通学路点検】」のように冒頭にタグをつける
- ノート機能に「決定事項」「年間スケジュール」をピン留め
- アルバムに引き継ぎ用の写真・資料を保存

#### ② 遊び場開放グループ

| 区分 | メンバー | 備考 |
|---|---|---|
| 遊び場委員 | 委員長1・会計2・指導員3 | 6名 |
| 執行委員長 | 執行委員会の委員長 | 全体把握のため参加 |
| 執行副委員長 | 執行委員会の副委員長 | 遊び場副委員長を兼務 |
| PTA役員 | 会長 or 副会長（少なくとも1名） | メイングループと同じ人 |
| 支援Co | 学校支援コーディネーター | 指導員募集・地域連携 |

**用途：** 開放日のシフト調整、当日の連絡（天候による中止判断等）、指導員との調整、日誌の共有、備品・消耗品の相談

**運用ルール：**
- 急ぎの現場判断（天候中止・事故対応等）はここで即時共有
- 月次の活動まとめは遊び場委員長がメイングループにも転記して報告
- ノート機能にシフト表・指導員連絡先・緊急時対応フローを保管

#### グループ間の情報連携

遊び場で起きたことがメイングループに伝わらない「サイロ化」を防ぐため、以下のルールを設ける。

- **月1回：** 遊び場委員長がメイングループに活動報告（利用状況・特記事項・次月の予定）を投稿
- **即時：** 事故・トラブル・中止判断はメイングループにも同時共有
- **執行委員長は両方に参加：** 全体の状況を常に把握し、必要に応じて橋渡し

### 4-2. PTA役員・支援コーディネーターの関わり方

PTA役員と支援コーディネーターは「常時見守り・必要時に介入」のスタンスで関わる。日常的な業務は執行委員会が自律的に回し、役員やCoは以下の場面で動く。

**PTA役員の関わり：**

| 場面 | 具体的なアクション |
|---|---|
| 年度初め | 学校との覚書内容を執行委員長に説明・共有 |
| 予算が必要なとき | 事務経費の申請を受けて、学校コミュニティ基金から支出を決裁 |
| 学校との調整が必要なとき | 副校長との連絡を仲介（委員長からの相談を受けて） |
| トラブル発生時 | 判断が難しい問題（人間関係、学校との方針齟齬など）の相談先 |
| 年度末 | 活動報告を受領、次年度の募集を支援 |

**学校支援コーディネーターの関わり：**

| 場面 | 具体的なアクション |
|---|---|
| 遊び場開放 | 地域との連携が必要な場面での橋渡し（指導員の募集支援など） |
| 通学路点検 | 地域の情報提供（危険箇所の住民からの声など） |
| 人手不足のとき | おやフェス等の学校コミュニティプロジェクト参加者への呼びかけ支援 |
| 全般 | 保護者が負担を感じている兆候があれば役員会にフィードバック |

## 5. 遊び場委員と遊び場開放運営委員会の関係整理

遊び場開放に関しては、PTA執行委員会と区の委託先である遊び場開放運営委員会の2つの組織が関わる。執行委員会の遊び場委員6名＋執行副委員長が、運営委員会の名簿上でどの職名に対応するかを整理する。

### 5-1. 運営委員会の役員構成（規約より）

遊び場開放運営委員会は定員15名以内。役員は委員の互選で選出する。

| 職名 | 責務 |
|---|---|
| 委員長 | 運営委員会を代表し会務を総理する |
| 副委員長 | 委員長を補佐し、委員長に事故あるときは職務を代理する |
| 会計事務担当者 | 遊び場開放の経費（区の委託料）を管理する |
| 会計監査 | 会計の経費管理を監査する |

委員の構成枠：PTA代表 / 地域内青少年委員 / 青少年地区委員 / 地域内社会教育関係団体代表 / 自治会代表 / 指導員 / 学校関係職員 / 区民活動に理解と熱意のある者 / BOP / その他必要と認めた者

### 5-2. PTA側の人員と運営委員会名簿の対応

| PTA執行委員会での役割 | 運営委員会名簿上の職名 | 備考 |
|---|---|---|
| 遊び場委員長（1名） | **委員長** | 運営委員会を代表。区教委への報告義務を負う |
| 執行副委員長（兼務） | **副委員長** | 委員長補佐・職務代理。執行副委員長が兼務 |
| 遊び場会計（2名のうち1名） | **会計事務担当者** | 委託料の管理・執行を担当 |
| 遊び場会計（2名のうち1名） | **会計監査** | 会計の経費管理を監査 |
| 遊び場指導員（3名） | **委員（指導員兼務）** | 開放日の現場で児童を指導。日誌を作成・提出 |
| 執行委員長 | **委員（PTA代表枠）** | 第3条1号。運営委員会に出席 |

### 5-3. 運営委員会の報告義務（委員長が負う）

## 6. 執行委員 選出ルール

### 6-1. 引き継ぎ（遊び場委員）

5月9日（土）10:00〜12:00（終わり次第解散）に引き継ぎを実施予定。

### 6-2. 卒対免除との関係（前提）

現3年生が6年生になった際の卒対くじ引きで免除対象となる。ただし、免除ルールの最終決定権は6年時のクラスにある。

### 6-3. 選出の手順

**ステップ1：立候補の募集**
保護者会当日にその場で立候補を受け付ける。加えて、後日Google Formsでも追加の立候補を受け付ける。

**ステップ2：立候補で人数が充足した場合**
→ 選出完了。保護者に結果を通知する。

**ステップ3：立候補で人数が不足する場合 → くじ引き**
以下の手順でオンラインくじを実施する。

1. **くじ対象者の確定** — 3年生保護者全体から、以下を除外してくじ対象者リストを作成する。
   - 立候補済みの方
   - 1・2年生で委員・役員を経験した方
2. **くじの実施** — 委員長がGoogleスプレッドシートのランダム関数（`RAND()`）等を用いて抽選を行う。
3. **透明性の確保** — くじの実施過程を画面録画等で記録し、希望者に共有できるようにする。
4. **結果の通知** — くじ結果をGoogle Forms等で個別に通知する。

### 6-4. 保護者会での対応（案B：後日オンラインくじ方式）

保護者会はクラス全員が集まっての開催となるが、くじはその場では行わず後日オンラインで実施する。理由は以下のとおり。

- **公平性：** 保護者会に出席できなかった方がくじを免れる不公平を防ぐ
- **正確性：** 免除対象者（1・2年で委員・役員経験者）の確認を事前にきちんと行える
- **心理的負担：** その場のプレッシャーでの選出を避けられる

**保護者会当日の流れ：**
1. 執行委員会の体制と役割を説明（スライド1枚を使用）
2. 選出ルール・卒対免除の前提を説明
3. その場で立候補を募集
4. 「追加の立候補はGoogle Formsでも受け付けます」と案内
5. 立候補で不足した場合のくじ引きの仕組み（オンライン・1,2年経験者除外）を説明
6. 質疑応答

## 7. 旗振り管理／通学路点検担当の位置づけ

旗振り管理と通学路点検は2名が兼務しあう体制とする。どちらの業務も活動頻度が低い（旗振り：学期に1回、通学路：年1回）ため、専任で分ける必要がなく、2名で両方の業務を分担・カバーし合う。

**旗振り管理：** 1年生の旗振り当番のアサインを管理する役割。実際に旗を振るのは1年生保護者であり、執行委員（3年生保護者）が管理・調整を担う。

現行の運営細則第4条では旗振り委員会の対象を「1年生保護者」としているが、これは「旗を振る当番の対象」として読める。管理を執行委員会（3年生）が担う構造にする場合、覚書または細則の補足で以下を明確にするのが望ましい。

- 旗振り当番：1年生保護者（＝実際に旗を振る人）
- 旗振り管理：執行委員会（3年生保護者）が当番のアサイン・管理を行う

**通学路点検：** 副校長からの依頼に基づき、通学路の危険箇所について保護者からの意見を徴収・とりまとめする。年1回の業務。

## 8. 12名確保の現実性

3年生保護者74名に対して12名の執行委員を選出する場合、**約16%（6.2人に1人）** の割合。

| 役割 | 人数 | 活動頻度 | なり手の見つけやすさ |
|---|---|---|---|
| 委員長 | 1名 | 通期 | 最もハードルが高い |
| 副委員長 | 1名 | 通期（遊び場副委員長兼務） | 委員長に次ぐ |
| 行事受付担当 | 2名 | 年数回 | 比較的見つけやすい |
| 旗振り管理／通学路点検担当 | 2名 | 旗振り：学期1回、通学路：年1回 | 比較的見つけやすい |
| 遊び場委員長 | 1名 | 通期 | ハードルが高い |
| 遊び場会計 | 2名 | 通期 | 中程度 |
| 遊び場指導員 | 3名 | 開放日のシフト | シフト制なので比較的見つけやすい |

行事受付2名・旗振り管理／通学路点検2名の計4名は活動頻度が低いので立候補が出やすい。課題は委員長・副委員長・遊び場委員長の「通期で責任を負うポジション」3名の確保。
