```
お子さまの情報と、入会についてのご意思をご回答ください。
```

お子さまのクラス

【設定メモ】「設定」→「プレゼンテーション」→「確認メッセージ」に入力

```
ご回答ありがとうございました。

ご不明な点がございましたら、PTA本部までお気軽にお問い合わせください。

※回答内容の変更をご希望の場合は、PTA本部までご連絡ください。
```

## Googleフォームの推奨設定

| 設定項目 | 推奨値 | 理由 |
|---------|--------|------|
| メールアドレスの収集 | OFF | ハードルを下げる。特定はクラス＋氏名で可能 |
| 回答のコピーを回答者に送信 | OFF（上記OFFのため不可） | — |
| 回答の編集を許可 | OFF | 意思表示の確定性を担保 |
| 1回のみ回答に制限 | OFF | Googleアカウント必須になるため避ける |
| 進行状況バー | ON | セクションが2つあるため表示推奨 |
| シャッフル | OFF | 順序に意味があるため固定 |

---

## 運用メモ

- **回答期限**：配布日から1〜2週間を目安に設定。期限はフォーム説明文またはお便りに明記する。
- **未回答への対応**：期限後にクラス名簿と突合し、未回答の方にはお便りまたは連絡帳で個別にリマインド。
- **回答データの管理**：スプレッドシートに自動連携し、クラス別の入会率を集計。個人情報として適切に管理する。
- **入会しなかった方への対応**：年度途中での入会希望にも対応できるよう、別途受付手段を用意しておく。
